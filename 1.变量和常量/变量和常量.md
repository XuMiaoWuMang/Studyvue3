# JavaScript 中的变量与常量

## 变量

### 1. 声明变量

JavaScript 中有三种声明变量的方式：`var`、`let` 和 `const`（虽然 `const` 主要用于声明常量，但它也是声明变量的一种方式）。

```javascript
var a = 0;        // 使用 var 声明变量
let b = '1';      // 使用 let 声明变量
console.log(a, b);
```

- `var`：是 JavaScript 早期版本的变量声明方式，函数作用域
- `let`：ES6 引入的变量声明方式，块级作用域，推荐使用

### 2. 修改变量

变量可以在声明后修改其值：

```javascript
a = 1;        // 修改 var 声明的变量
b = '2';      // 修改 let 声明的变量
console.log(a, b);
```

## 常量

### 1. 声明常量

使用 `const` 关键字声明常量：

```javascript
const c = 3;
const d = '4';
```

### 2. 常量的特性

- 常量一旦声明，其值不能被修改
- 尝试修改常量的值会导致错误：

```javascript
c = 4; // 报错
d = '5'; // 报错
```

### 3. 常量引用类型的特殊情况

虽然常量的值不能被修改，但对于引用类型（如数组和对象），可以修改其内容：

#### 数组操作

```javascript
const arr = [1, 2, 3];
arr[0] = 4;          // 修改数组元素
console.log(arr);    // 输出: [4, 2, 3]

const arr1 = [1, 2, 3];
arr1.push(4);        // 添加元素
console.log(arr1);   // 输出: [1, 2, 3, 4]
arr1.shift();        // 删除第一个元素
console.log(arr1);   // 输出: [2, 3, 4]
```

#### 对象操作

```javascript
const obj = {
    name: '张三',
    age: 18
};
obj.name = '李四';    // 修改属性值
console.log(obj);    // 输出: {name: "李四", age: 18}

const obj1 = {
    name: '张三',
    age: 18
};
obj1.name = '李四';    // 修改已有属性
obj1.age = 19;        // 修改已有属性
obj1.gender = '男';    // 添加新属性
console.log(obj1);    // 输出: {name: "李四", age: 19, gender: "男"}
```

## 总结

1. **变量**：
   - 使用 `var` 或 `let` 声明
   - 可以在声明后修改其值
   - `let` 是更现代的声明方式，推荐使用

2. **常量**：
   - 使用 `const` 声明
   - 声明后不能修改其值
   - 对于引用类型（数组和对象），可以修改其内容但不能重新赋值

3. **最佳实践**：
   - 优先使用 `const` 声明不会改变的值
   - 使用 `let` 声明会改变的值
   - 避免使用 `var`，除非有特殊需求
